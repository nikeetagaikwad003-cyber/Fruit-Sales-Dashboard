# Fruit Sales Dashboard (Power BI)

## Project Overview
This project presents an interactive Power BI dashboard to analyze fruit sales performance across different categories, regions, and time periods. The dashboard helps identify sales trends and high-performing products.

##  Tools Used
- Power BI
- Data Modeling
- DAX (basic measures)

##  Key Insights
- Total sales by fruit category
- Regional sales performance
- Monthly and yearly sales trends
- Identification of top-selling fruits

##  Dashboard Features
- Interactive visuals
- Slicers for filtering data
- Clean and user-friendly layout
